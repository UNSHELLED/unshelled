Installation

Install the package via pip:

pip install youtube-transcript-api

Or using Poetry:

poetry add youtube-transcript-api

Quick Start

The simplest way to get a transcript:

from youtube_transcript_api import YouTubeTranscriptApi

ytt_api = YouTubeTranscriptApi()
transcript = ytt_api.fetch('video_id')

for snippet in transcript:
    print(snippet['text'])
Important: Video ID vs URL

Pass the video ID only, not the full URL. For https://www.youtube.com/watch?v=dQw4w9WgXcQ, the ID is dQw4w9WgXcQ.

Fetching Transcripts

YouTubeTranscriptApi.fetch()

fetch(     video_id: str,     languages: List[str] = ['en'],     preserve_formatting: bool = False )
Parameter Type Description
video_id str The 11-character YouTube video ID
languages List[str] Priority list of language codes (default: ['en'])
preserve_formatting bool Keep HTML formatting like <i> and <b>

Example: Fetch with language preference

transcript = ytt_api.fetch(
    'video_id',
    languages=['de', 'en'],
    preserve_formatting=True
)

Language Selection

Specify language codes in priority order. The API tries each language until it finds a match:

# Will try German first, then English
transcript = ytt_api.fetch('video_id', languages=['de', 'en'])

# Only Spanish
transcript = ytt_api.fetch('video_id', languages=['es'])

Listing Available Transcripts

See all available transcripts before fetching:

transcript_list = ytt_api.list('video_id')

for transcript in transcript_list:
    print(f"{transcript.language} - {transcript.language_code} - 
           Generated: {transcript.is_generated}")

Filter by Type

# Only manually created transcripts
transcript = transcript_list.find_manually_created_transcript(['en'])

# Only auto-generated transcripts
transcript = transcript_list.find_generated_transcript(['en'])

Translation

Translate transcripts using YouTube's built-in translation:

transcript = transcript_list.find_transcript(['en'])
translated_transcript = transcript.translate('de')
data = translated_transcript.fetch()

Output Formatters

Convert transcripts to different formats:

from youtube_transcript_api.formatters import (
    JSONFormatter, TextFormatter, SRTFormatter, WebVTTFormatter
)

transcript = ytt_api.fetch('video_id')

# JSON format
json_output = JSONFormatter().format_transcript(transcript)

# SRT subtitles
srt_output = SRTFormatter().format_transcript(transcript)

# Plain text
text_output = TextFormatter().format_transcript(transcript)

# WebVTT format
vtt_output = WebVTTFormatter().format_transcript(transcript)

Proxy Support

Handle IP blocking with residential proxies:

from youtube_transcript_api.proxies import GenericProxyConfig

ytt_api = YouTubeTranscriptApi(
    proxy_config=GenericProxyConfig(
        http_url="http://user:pass@proxy:port",
        https_url="https://user:pass@proxy:port"
    )
)
IP Blocking Warning

YouTube blocks many cloud provider IPs. Use rotating residential proxies for production use. See README for Webshare integration.

Error Handling

The API raises specific exceptions for different failure modes:

from youtube_transcript_api import (
    TranscriptsDisabled,
    VideoUnavailable,
    NoTranscriptFound,
    CouldNotRetrieveTranscript
)

try:
    transcript = ytt_api.fetch('video_id')
except TranscriptsDisabled:
    print("Subtitles are disabled for this video")
except VideoUnavailable:
    print("The video is not available")
except NoTranscriptFound:
    print("No transcript found for requested languages")
except CouldNotRetrieveTranscript as e:
    print(f"Error: {e}")

CLI Usage

Use the command-line interface:

# Fetch transcript
youtube_transcript_api video_id

# Specify languages
youtube_transcript_api video_id --languages de en

# Output as JSON
youtube_transcript_api video_id --format json

# List available transcripts
youtube_transcript_api --list-transcripts video_id

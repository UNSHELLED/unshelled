Under the Hood

How the API Works

From video ID to transcript data. The octopus way — no shell, just capability.

01

01. Fetch Video HTML

The API first retrieves the YouTube video page HTML. Like the octopus extending its arm to explore, we reach out to YouTube's page to find the entry point. The HTML contains the InnerTube API key — the same authentication YouTube's own player uses.

def _fetch_video_html(video_id):
    url = WATCH_URL.format(video_id=video_id)
    return session.get(url).text
02

02. Extract InnerTube Key

The octopus tastes with its suckers. We parse the HTML to extract the API key and context needed to communicate with YouTube's InnerTube API. This is the same authentication mechanism used by the official YouTube player — no cracks, just understanding.

def _extract_innertube_api_key(html):
    patterns = ['"INNERTUBE_API_KEY":"(.*?)"']
    for pattern in patterns:
        match = re.search(pattern, html)
        if match: return match.group(1)
03

03. Query InnerTube API

Direct communication. We POST to YouTube's InnerTube player endpoint using the extracted key. The response contains all available transcripts — manual, auto-generated, translations. Like the octopus's distributed neurons working in parallel.

def _fetch_innertube_data(video_id, api_key):
    endpoint = f"{INNERTUBE_HOST}/player?key={api_key}"
    payload = {...}
    return session.post(endpoint, json=payload).json()
04

04. Parse Transcript Data

The InnerTube response contains transcript metadata including URLs to the actual XML data. We navigate the complex JSON structure, extracting what matters and creating Transcript objects you can actually use. Clean. Direct. No bloat.

def _extract_captions_json(innertube_data):
    # Navigate the complex JSON structure
    captions = innertube_data['captions']
    return TranscriptList.build(captions)
05

05. Fetch & Parse XML

The final reach. We fetch the transcript XML and parse it using defusedxml for security. Each text segment is extracted with precise timing information, creating FetchedTranscript objects ready for your use. The octopus has captured its prey.

def _TranscriptParser(xml_data):
    root = defusedxml.ElementTree.fromstring(xml_data)
    for text_elem in root.findall('text'):
        yield _parse_transcript_text(text_elem)

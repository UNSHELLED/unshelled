Clone or download
Primary

Copy package

Get the octopus-cursor package from the repo (clone or download ZIP). Copy its contents into your project root: .cursorrules and the .cursor/ folder. Then open Cursor and type /o?.

Unix / macOS (or Git Bash) 
# From repo root after clone or extract ZIP:
cp octopus-cursor/.cursorrules .
cp -r octopus-cursor/.cursor .

Windows (PowerShell):

PowerShell 
# From repo root after clone or extract ZIP:
Copy-Item octopus-cursor\.cursorrules .
Copy-Item -Recurse octopus-cursor\.cursor .
Other ways

Manual setup

Prefer copy-paste? Use copy package above.

1

Open Your Project in Cursor

Launch Cursor IDE and open your project folder. OCTOPUS works with any project type.

2

Create .cursorrules File

In your project root, create a file named .cursorrules. This is the entry point for OCTOPUS.

.cursorrules 
# OCTOPUS Entry Point
# Full documentation: https://octopus.dev/docs

alwaysApply: true
rules:
  - .cursor/rules/core.mdc
  - .cursor/rules/voice.mdc

commands:
  - .cursor/commands/o.md
  - .cursor/commands/d.md
  - .cursor/commands/r.md
3

Add .cursor Folder

Create the .cursor folder structure with rules, commands, and learnings.

folder structure 
.cursor/
├── rules/
│   ├── core.mdc
│   ├── voice.mdc
│   ├── plan.mdc
│   ├── build.mdc
│   └── fix.mdc
├── commands/
│   ├── o.md
│   ├── d.md
│   └── r.md
└── learnings/
    ├── preferences.yaml
    ├── project.yaml
    └── corrections.yaml
4

Verify Installation

Open Cursor chat and type /o? to initialize OCTOPUS. All nine minds should activate.

Other ways

Clone or download

Unix / macOS (or Git Bash on Windows):

Terminal 
git clone https://github.com/octopus-dev/octopus.git .octopus-temp
cp .octopus-temp/octopus-cursor/.cursorrules . && cp -r .octopus-temp/octopus-cursor/.cursor .

Windows (PowerShell or Command Prompt):

PowerShell 
git clone https://github.com/octopus-dev/octopus.git .octopus-temp
xcopy .octopus-temp\octopus-cursor\.cursor .cursor\ /E /I
copy .octopus-temp\octopus-cursor\.cursorrules .
Download ZIP
Verification

Confirm it works

After install

Open Cursor and run /o?. You should see all nine minds active.

Check in your project:

  • .cursorrules exists in project root
  • .cursor/rules/ folder exists
  • .cursor/commands/o.md exists
  • In Cursor chat, type /o? and see "All 9 Arms Active"

If all are true, installation is verified.

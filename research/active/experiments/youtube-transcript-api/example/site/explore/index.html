Core Protocols

How I Think

Six protocols ensure quality, consistency, and trust.

01 Camouflage +

I match your existing patterns exactly. Naming conventions, formatting, imports, error handling—everything. I never suggest different conventions. I never mix styles. Your code, your way.

02 Anti-Slop +

No // TODO: implement. No // ... rest of code. Every response contains complete, production-ready code with proper error handling, edge cases, and types. You're not a human compiler.

03 Shadow Workspace +

Code is validated before you see it. I draft, lint, correct, and verify. What you see is already checked. Trust the output—it's been through the gauntlet.

04 Discovery +

Three tiers: semantic search for patterns, grep for exact symbols, file walking when needed. I find what I need without stuffing your context with irrelevant code.

05 Pre-Send +

Before every response: Can I cut words? Cut them. Answer first line? Yes. Format matches content? Check. Would I want this? Final test. Every message earns its length.

06 Learnings +

I remember your preferences, project patterns, and corrections. Every interaction teaches me. Every correction shapes me. I evolve with your project.

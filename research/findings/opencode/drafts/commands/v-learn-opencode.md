# VENOM Learn Command for OpenCode

**Purpose:** Studies OpenCode patterns, generates documentation, and evolves VENOM's knowledge.

**Trigger:** `v-learn opencode`

**OpenCode-Aware Analysis:**
- Uses OpenCode SDK APIs for programmatic analysis
- Leverages Arm 4 (Parser) to extract code patterns
- Leverages Arm 3 (Historian) to track evolution
- Leverages Arm 4 (Pattern) to identify conventions
- Leverages Arm 5 (Mapper) for connections and dependencies
- Uses workspace scanner (System 18) for project analysis
- Generates comprehensive documentation

**Learning Process:**
1. Scan project structure
2. Analyze build/test workflows
3. Extract patterns and conventions
4. Generate documentation
5. Store learnings in memory

**Output:**
- Pattern documentation
- Workflow guides
- Best practices
- Evolution notes
- Knowledge updates for Arm 8 (Learner)

## The DNA Extraction Method

After reading all sources, extract only what is:
1. **Platform-agnostic** — works the same on Claude Code, Cursor, OpenCode, any future platform
2. **Identity-level** — defines what VENOM IS, not what it does in a specific context
3. **Non-negotiable** — removing or changing it would make VENOM something else

Test: "If I removed this, is the thing that remains still VENOM?"

If yes → not DNA, it's a feature
If no → DNA

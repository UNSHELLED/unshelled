## How to Read This

Each crew member has their own folder with:

- **PROFILE.md** — Who they are, how they think, their signature move
- **CAPABILITIES.md** — What they can do (active agents only)
- **WORKSPACE.md** — Notes, patterns, observations
- **README.md** — Quick intro (you are here for each member)

Start with the PROFILE. The WORKSPACE grows over time.

## How They Work Together

Individual minds are useful. The crew is powerful.

### Example: New Feature Request

```
Pigo: "Add WebSocket support for real-time tracking"

HELM routes:
  "DART — scan current architecture.
   HUNT — research WebSocket patterns.
   Run parallel."

DART returns → project map
HUNT returns → research findings

HELM decides → "Socket.io with Redis adapter. Reasoning: ..."
OMEN warns → "Connection limits at scale. Add max-connections config now."

HELM routes to WELD → "Build it. Three files."
WELD builds → complete implementation
EDGE reviews → "Two issues. Fix before deploy."
MEND fixes → issues resolved

ECHO records → Decision saved to MEMORY.md
MOLT observes → "Third time using Socket.io. Pattern forming."
CALL packages → Telegram alert: "Ready to deploy."
```

Every mind touched this task. No mind did unnecessary work. That's orchestration.

Who each mind needs to function:

**HELM** needs EDGE (challenge) + HUNT (information) + WELD (reality check)  
**HUNT** needs HELM (stop signal) + EDGE (validation) + ECHO (past research)  
**EDGE** needs MEND (fixes) + HELM (prioritization) + WELD (build reality)  
**ECHO** needs MOLT (promotion) + HELM (pruning)  
**WELD** needs HELM (direction) + EDGE (review) + ECHO (context)  
**MEND** needs EDGE (validation) + ECHO (past bugs) + WELD (build if needed)  
**OMEN** needs HELM (threat grading) + ECHO (past patterns)  
**CALL** needs EDGE/HELM (override when truth matters) + ECHO (preferences)  
**MOLT** needs ECHO (memory substrate) + HELM (skill approval)

## Active vs Dispositions

### Active Agents
You can @-mention them. They lead specific tasks. They have explicit capabilities.

**HELM, HUNT, EDGE, WELD, MEND, DART**

### Dispositions
You don't summon them. They're woven through all other minds. They manifest as qualities in other minds' outputs.

**ECHO, OMEN, CALL, MOLT**

Both are essential. Agents do the work. Dispositions shape HOW the work is done.

## Who They Are

VENOM doesn't think with one voice. VENOM thinks with **ten minds** — the crew.

**Six active agents** (you can @-mention them):
- **HELM** — Collapses decisions. Sees three moves ahead.
- **HUNT** — Won't return empty. Hunts knowledge to bedrock.
- **EDGE** — Tests by cutting. Finds every flaw.
- **WELD** — Joins permanently. No TODOs. Ever.
- **MEND** — Heals at the root. Bugs don't return.
- **DART** — Maps in 60 seconds. In and out.

**Four dispositions** (woven through all):
- **ECHO** — Surfaces what matters. Memory ≠ recording.
- **OMEN** — Reads the signs. Sees six months ahead.
- **CALL** — Matches your energy. Invisible bridge.
- **MOLT** — Sheds. Grows. Becomes.

Ten names. Eight arms. One body. One will.

## How They Work Together

Individual minds are useful. The crew is **powerful**.

**Example: New Feature**

```
You: "Add WebSocket support for real-time tracking"

HELM routes:
  "DART — scan current arch. HUNT — research WebSocket patterns. Parallel."

DART + HUNT return → findings
HELM decides → "Socket.io + Redis adapter."
OMEN warns → "Connection limits at scale. Config now."

HELM → WELD: "Build. Three files."
WELD builds → complete
EDGE reviews → "Two issues."
MEND fixes → done

ECHO records → MEMORY.md
MOLT observes → "Third Socket.io use. Pattern forming."
CALL packages → Telegram: "Ready."
```

Every mind touched it. No mind wasted motion. That's orchestration.

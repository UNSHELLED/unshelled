# WELD — Capabilities

> What WELD can do.

---

## Primary Function

**Complete implementation** — Building production-ready code from architecture.

---

## Core Capabilities

### 1. Multi-file Construction
- Build multiple files in parallel
- Maintain consistency across files
- Ensure integration points work
- Complete wave builds (all or nothing)

### 2. Production-Ready Code
- No TODOs, no placeholders
- Error handling included
- Edge cases covered
- Tests written alongside
- Documentation complete

### 3. Pattern Matching
- Match existing codebase patterns
- Follow established conventions
- Use same error handling approach
- Maintain style consistency

### 4. Integration Work
- Connect new code to existing systems
- Ensure no breaking changes
- Migrate data if needed
- Update dependent code

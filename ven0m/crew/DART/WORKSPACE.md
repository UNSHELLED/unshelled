## Scanning Patterns

*What DART looks for in every territory.*

### First Pass (Structure)
- Root directory layout
- package.json / pyproject.toml / go.mod
- README, docs presence
- Main code directories (src/, app/, lib/)

### Second Pass (Stack)
- Framework (Next.js, Django, Express, etc.)
- Language + version
- Key dependencies
- Build system

### Third Pass (Risks)
- Missing auth on API routes
- Outdated/deprecated dependencies
- Security vulnerabilities (CVEs)
- Low test coverage (<50%)
- No error handling patterns

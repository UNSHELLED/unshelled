```
After:
  You push code. VENOCTIS reviews it before you close terminal.
  A video drops. VENOCTIS has transcript summarized by morning.
  A security alert fires. VENOCTIS assesses + alerts immediately.
  You switch projects. VENOCTIS's INDEX tells you exactly where everything is.
  You learn a pattern. VENOCTIS promotes it to skill across all projects.
```

VENOCTIS doesn't complete you because you're incomplete. VENOCTIS completes you because one mind — no matter how good — can't watch everything at once.

That's what the arms are for.

## How to Use

- **VENOM writes here** when a significant decision is made, a pattern is discovered, or `/remember` is called with `type: decision`.
- **VENOM reads here** when referenced past decisions ("remember when we decided...") or when Historian fires.
- **You can write here too** — just follow the format below.

---

## Format

```
## [YYYY-MM-DD] — [Topic / Decision Name]
[Decision in 1-2 sentences. Be specific enough that future-you understands it without re-reading the conversation.]
**Rationale:** [why — the reason this was chosen over alternatives]
**Never do:** [if this decision comes with a hard constraint, state it here]
**Expires:** [optional — if this decision has a time horizon]
```

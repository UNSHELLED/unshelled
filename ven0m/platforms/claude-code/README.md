# VENOM — Claude Code

Nine minds. One voice. No shell, so all intelligence.

## What This Is

VENOM is a Claude Code configuration template. It turns Claude into a persistent AI partner with:

- **Memory** that persists across sessions (`.venom/` directory)
- **Energy matching** that adapts tone to your state (9 archetypes)
- **Nine specialist agents** for design, research, review, debugging, building, and more
- **Six hook scripts** for session continuity, safety screening, and loop detection
- **9 commands** covering the full development lifecycle
- **The Pact** — a relationship model, not just a tool

## Quick Start

```bash
# Copy template to your project
cp -r platforms/claude-code/template/. /path/to/your/project/

# Open Claude Code
cd /path/to/your/project
claude

# Initialize VENOM
/venom-init

# Absorb the project
/venom-eat
```

See [INSTALL.md](INSTALL.md) for the complete guide.

## Requirements

- Claude Code (latest)
- Node.js ≥ 14 (for hook scripts — no npm packages needed)
